## ◈ &nbsp; About Me

<img align="right" width="280" src="https://raw.githubusercontent.com/Potential17/Potential17/master/user%20(2).gif" />

I'm **Saman Fatima**, a data scientist from Pakistan and a Kaggle Legacy Grandmaster ranked **#191 of 62,137** globally (best ever: **#24**).

My approach is simple — start from the data, ask honest questions, and build something that actually answers them. I write notebooks the way I wish someone had written them for me: clearly, with the reasoning shown, and with results you can reproduce.

**What I work on:**

- 📊 &nbsp; Exploratory analysis and classical ML on tabular data
- 👁️ &nbsp; Deep learning for computer vision — CNNs, classifiers at 90%+ accuracy
- 🧠 &nbsp; NLP, sequence models, red-teaming walkthroughs
- 🤖 &nbsp; LLM pipelines and multi-agent systems (hackathon projects)
- 📈 &nbsp; Time-series forecasting and financial modelling

**Beyond building**, I mentor juniors in the data-science community and help aspiring Kagglers find their footing on the platform.
